# Server

Как пользоваться:
1) Запускаем программу
2) Выбираем сервер
3) Создаём второй экземпляр приложения
4) Во втрором выбираем клиента
5) Отправляем сообщение на сервер
6) От сервера получаем вместо ответа вопросительные знаки
7) С кайфом выходим
